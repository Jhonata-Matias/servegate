### 2. Rework tests — stop tautological assertions

**Files:**
- `serverless/tests/test_handler_qwen_workflow.py`
- `serverless/tests/test_handler_dispatch.py` (lines 70-71)

**Current pattern (tautological):**

```python

```

This test is worthless — the handler emits those strings because the test tells it to, and the test passes because the handler emits those strings. No external ground truth.

**Required pattern (choose one):**

**Option A — known-good reference fixture**
Commit a `serverless/tests/fixtures/comfyui-v{VERSION}-object-info.json` captured from a live ComfyUI `/object_info` endpoint. Test asserts every `class_type` emitted by the handler appears in that fixture. Refresh fixture on `COMFY_REF` bump.

**Option B — live validation (optional, slower)**

**Minimum acceptable:** Option A. Register the fixture-regeneration process in the Dockerfile or a `scripts/refresh-comfy-fixture.sh`.
