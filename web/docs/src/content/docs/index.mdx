<Badge text="Alpha" variant="caution" /> servegate is the canonical HTTP contract for generating images with **FLUX.1-schnell**, editing images with **Qwen-Image-Edit** (single- or multi-image), and generating text with **Gemma 4**. Images use async submit/poll; text uses streaming SSE on the same authenticated gateway.

Latest server: `gemma4-flux-serverless:0.2.1` (Qwen-Image-Edit 2509 with multi-image support).
Latest SDK: `@jhonata-matias/flux-client@0.4.0`.

## Why servegate

- **Async-first.** Submit/poll lets long inferences (cold start ~130s, warm ~7s) coexist with short ones behind one authenticated edge.
- **Three modalities, one credential.** Same API key proxies T2I (FLUX.1-schnell), i2i (Qwen-Image-Edit single/multi), and text (Gemma 4 via vLLM).
- **Open source.** Apache 2.0 / MIT, alpha-public on GitHub.
- **Cost-aware.** RunPod Serverless behind Cloudflare Workers; near-zero idle.
